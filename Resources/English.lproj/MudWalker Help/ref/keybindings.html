Keyboard Reference

This document covers special keyboard actions for text editing. Most of these only operate in the input panel.

Opt-Return, Shift-Return
Inserts a line break in the input panel without sending it immediately.
Control-P, Cmd-Up, 'Prev' function key
Selects the previous history entry in the input panel. Cmd-Opt-Up selects the first.
Control-N, Cmd-Down, 'Next' function key
Selects the next history entry in the input panel. Cmd-Opt-Down selects the last.
Control-U
Deletes all text.
'Scroll Lock' function key or 'F14' function key
Toggles the Scroll Lock function for output displays. Note that some systems use F14 for monitor contrast control, in which case you can hold down any modifier key to use it for this shortcut.
