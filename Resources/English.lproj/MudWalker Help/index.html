MudWalker Help

There isn't much here. My excuse: I don't know what documentation would be useful, rather than simply clutter that restates the obvious.

So: the only information you'll find here is what I've written because I either felt it specifically needed documenting or was requested. If there is something you need help with, or feel needs further documentation, feel free to ask on the MudWalker mailing list, or contact me by one of the methods listed on my web site.

Known Problems

  • Logs of all sessions are automatically saved into Library/Logs/MudWalker Log document-name.txt. This is not configurable.
  • The "Expert Editor" settings pane doesn't actually allow editing anything.
  • The "Key Macros" settings pane does not provide a nice way to specify keystrokes.

Implementation Note

Connection documents are saved as serialized objects using NSArchiver. This means, for example, you can read a document using F-Script and examine or modify it.
